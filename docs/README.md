# AI CLI Integration Plugin

## Обзор

Этот плагин для IntelliJ IDEA предоставляет интеграцию с популярными AI инструментами командной строки, включая Codex CLI, Claude Code, и другие модели искусственного интеллекта. Плагин значительно упрощает процесс разработки, позволяя разработчикам использовать возможности AI прямо в IDE.

## Цель проекта

Создание единого интерфейса в IntelliJ IDEA для работы с различными AI инструментами, что позволит:

- Повысить продуктивность разработчиков
- Улучшить качество кода через AI-ассистированную разработку  
- Интегрировать современные AI инструменты в привычный workflow
- Обеспечить единообразный опыт работы с разными AI моделями

## Поддерживаемые AI инструменты

### Приоритетные для реализации:
- **Claude Code** - официальный CLI от Anthropic
- **Codex CLI** - инструмент на основе OpenAI Codex
- **GitHub Copilot CLI** - интеграция с gh copilot

### Планируемые для будущих версий:
- **Cursor AI** - интеграция с Cursor
- **Codeium** - бесплатная альтернатива Copilot
- **Tabnine** - AI помощник для кода
- **Amazon CodeWhisperer** - AWS AI coding assistant

## Архитектура

Плагин построен на основе IntelliJ Platform и использует:

- **Tool Window** - основной интерфейс взаимодействия
- **Action System** - интеграция в меню и toolbar
- **External Tools Integration** - для запуска CLI команд
- **Editor Integration** - контекстные действия в редакторе
- **Settings/Configurable** - конфигурация AI инструментов

## Быстрый старт

1. Установите необходимые AI CLI инструменты
2. Настройте плагин через Settings → Tools → AI CLI Integration
3. Используйте Tool Window или контекстные меню для взаимодействия с AI

## Требования

- IntelliJ IDEA 2024.3.6+ (или совместимые IDE)
- Java 21+
- Установленные AI CLI инструменты (Claude Code, Codex CLI, etc.)
